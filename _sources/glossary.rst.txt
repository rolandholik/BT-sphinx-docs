.. glossary::

   COE
    Context Of Execution.

   TO
    Trust Orchestrator.

   TMA
    Trusted Modeling Agent.

   TPM
    Trusted Platform Module.
    A hardware security chip that provides cryptographic keys, secure storage, and platform integrity measurements.

   TSEM
    Trusted Security Event Modeling.

   LSM
    Linux Security Module.
    A kernel framework that allows security modules (like AppArmor, SELinux, or TSEM) to enforce access control policies.

   HIDS
    Host Intrusion Detection System.
    A security system that monitors a host machine for suspicious activity, policy violations, or intrusions.

   OCI 
    Open Container Initiative.
    A standards body defining open specifications for container images and runtimes.

   FQDN
    Fully Qualified Domain Name.
    A complete domain name that specifies a host’s exact location in the DNS hierarchy.

   MQTT
    Message Queuing Telemetry Transport.
    A lightweight publish/subscribe messaging protocol designed for IoT devices and low‑bandwidth networks.

   PCR
    Platform Configuration Register
    A TPM register that stores cryptographic measurements of system state for attestation.

   SGX
    (Intel) Software Guard Extensions.
    Intel CPU extensions that allow applications to run code and store data inside secure hardware enclaves.

   IPC
    Inter-Process Communication.
    Mechanisms that allow processes to exchange data or signals.

   IP
    Internet Protocol

   SSH
    Secure shell
    The Secure Shell Protocol is a cryptographic network protocol for operating network services securely over an unsecured network.
